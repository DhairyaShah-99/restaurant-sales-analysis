# Restaurant Sales Analysis (Power BI)

This project analyzes restaurant sales data using **Power BI** to uncover insights about revenue performance, customer behavior, operational efficiency, and restaurant-level performance.

The dashboard provides a comprehensive view of restaurant sales through multiple analytical perspectives including **city performance, restaurant comparison, customer trends, delivery efficiency, and monthly sales patterns**.

# Dashboard Overview

The Power BI dashboard is designed with **multiple analytical views** that allow users to explore sales performance across different dimensions.

Users can interact with the dashboard using filters for:

- Month
- City
- Restaurant Name

Key performance indicators displayed in the dashboard include:

- **Net Revenue:** 40.52M
- **Total Orders:** 86,874
- **Average Order Value (AOV):** 466.34

# Key Business Insights

## City Performance
- Delhi generates the **highest order volume and revenue**
- Bangalore and Chennai also contribute significantly to total sales
- Mumbai shows comparatively **lower order volume**

## Restaurant Performance
- Restaurants such as **Barbeque Nation and Domino’s generate strong revenue**
- Some restaurants receive high order counts but comparatively lower revenue
- This helps identify **high-performing and underperforming outlets**

## Customer Behavior
- Customer distribution varies across cities
- Certain cities demonstrate **higher engagement and order activity**

## Delivery Performance
- Average delivery time varies slightly across cities
- Bangalore shows the **highest delivery time**, while Mumbai has the **lowest**

## Ratings & Discounts
- Restaurant ratings vary between brands
- Discount patterns indicate how different restaurants use pricing strategies to attract customers

## Monthly Trends
- Sales and customer activity increase significantly between **July and August**
- Discounts appear to decrease over time, indicating potential demand stability

# Dashboard Features

The dashboard includes multiple analytical visuals such as:

- **KPI Cards for Revenue, Orders, and Average Order Value**
- **City-wise Revenue Map**
- **Restaurant-wise Revenue vs Orders Analysis**
- **Customer Count by City**
- **Delivery Time Analysis**
- **Restaurant Rating Comparison**
- **Discount Analysis**
- **Monthly Sales Trend**
- **Customer Growth by Month**

These visualizations provide a **holistic understanding of restaurant sales operations and customer patterns**.
