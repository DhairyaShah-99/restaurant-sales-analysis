# Restaurant Sales Analysis (Power BI)

This project analyzes restaurant sales data using Power BI to identify revenue trends, popular menu items, and customer ordering patterns.

## Key Insights

- Total revenue generated from restaurant sales
- Most popular menu items
- Sales trends over time
- Customer ordering patterns

## Tools Used

Power BI  
Data Visualization  
Data Analysis  

## Dataset

The dataset contains restaurant sales information including menu items, order quantities, and revenue.
